# Employee Portal

Mobile-responsive web application where employees practice procedures independently with AI-guided coaching.

## Purpose

- Display step-by-step instructions from structured skill units
- Capture execution via device camera for real-time verification
- Provide immediate feedback on missed steps, incorrect actions, and safety compliance
- Track individual practice sessions and readiness scores

## Planned Tech

- Mobile-responsive web (React / Next.js)
- Browser camera API for execution capture
- Real-time feedback UI optimized for one-handed mobile use

## Dependencies

- `packages/shared` — skill unit schema and types
- `services/api` — session management, verification, readiness scoring

## Status

Not yet implemented. Scaffold planned for Phase 4.
