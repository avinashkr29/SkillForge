# Manager Portal

Web analytics dashboard giving supervisors visibility into workforce readiness and execution quality.

## Purpose

- View individual and team readiness scores
- Identify common execution gaps across the workforce
- Monitor compliance status and training progress
- Drill down from team metrics to individual session history

## Planned Tech

- Web dashboard (React / Next.js)
- Charts and tables for readiness analytics
- Filtering by team, skill unit, time range, and compliance status

## Dependencies

- `packages/shared` — skill unit schema and types
- `services/api` — readiness metrics, session history, gap analysis

## Status

Not yet implemented. Scaffold planned for Phase 6.
