# LEGO Assembly Skill: Yellow-Purple-Orange Tower

## Overview
This skill teaches employees how to assemble a LEGO tower by stacking colored blocks in the sequence: Yellow → Purple → Orange.

## Learning Objectives
- Identify colored LEGO blocks correctly
- Stack blocks in the correct sequence
- Verify each step before moving to the next block

## Skill Steps

### Step 1: Prepare the Base
- **Action**: Place the yellow block on the work surface
- **Verification**: Confirm the yellow block is stable and ready
- **Expected Duration**: 15 seconds

### Step 2: Add Purple Block
- **Action**: Take the purple block and place it on top of the yellow block
- **Verification**: Confirm the blocks are aligned and secure
- **Expected Duration**: 20 seconds

### Step 3: Add Orange Block
- **Action**: Take the orange block and place it on top of the purple block
- **Verification**: Confirm all three blocks are stacked correctly and the tower is stable
- **Expected Duration**: 20 seconds

## Success Criteria
- All blocks are in the correct sequence (Yellow → Purple → Orange)
- The tower is stable and does not fall
- All blocks are properly aligned

## Tools & Resources
- YOLO object detection for block identification and sequencing verification
- Video demonstration available in `sample_videos/`

## Estimated Time to Complete
Approximately 1 minute for experienced learners
